# 基于 PySide6 的终末地武器基质 app（自用）  

**目前实现了两种功能**
1. 规划武器基质刷取计划：根据希望刷取的武器基质计算最高效率的刷取方式，优先级按**目标武器基质数量**与顺**带可刷取的武器基质数量**对**词条与副本选择方式**进行排序。
2. 检查某基质是否有适用武器：根据武器基质词条判断当前版本是否有可使用的武器。

ps: 额外实现了仓检，但需要手动更改 inventory.json 中的值，weapon_possessed 为是否拥有对应武器，weapon_core_possessed 为是否拥有适配的武器基质；对于值，已拥有为 true，未拥有为 false。仓检功能需通过名为**检查仓储**的 checkbox 勾选。
